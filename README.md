# Who am I, Go? 

Simple HTTP project written in [Go](https://golang.org/) 

## API REST

### Response

- **me**: hostname (container ID for example)
- **timestamp**: execution start time
- **duration**: execution duration in microseconds 

```json
{
  "me": "RJ-MB2680.lan",
  "timestamp": "2019-02-12T10:00:00.000Z",
  "duration": 
}
```
